# AI-Based Brain State Recognition

A machine learning system that detects human brain and eye states using EEG/EOG signals. The project applies advanced signal processing techniques and classification models to accurately identify patterns such as blinking and eye movement.

---

## 📌 Overview
This project focuses on analyzing EEG/EOG signals to recognize brain states. It includes data preprocessing, feature extraction, and training multiple machine learning models to achieve high accuracy.

---

## 🚀 Features
- EEG/EOG signal processing
- Feature extraction using EMD (Empirical Mode Decomposition) and DWT (Discrete Wavelet Transform)
- Multiple ML models implementation:
  - Random Forest
  - Logistic Regression
  - Support Vector Machine (SVM)
  - Multi-Layer Perceptron (MLP)
- Achieved up to **98% accuracy** (best model: MLP)
- Custom dataset creation (500 samples)
